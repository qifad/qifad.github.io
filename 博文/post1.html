如何创建自定义物品和合成配方 - 泰拉瑞亚 Mod 制作教程

在这个教程中，我们将介绍如何使用 Tmodloader 创建自定义物品和合成配方。

using Terraria;
using Terraria.ID;
using Terraria.ModLoader;

namespace TestMod.Items
{
 public class A : ModItem
 {
 // 设置默认值和添加合成配方的代码将在后续步骤中进行解释
 }
}
C#

在 `SetDefaults` 方法中，你可以设置物品的各种属性，例如伤害、尺寸、使用时间等。

// 设置物品属性
public override void SetDefaults()
{
    Item.damage = 50; // 物品伤害
    Item.DamageType = DamageClass.Melee; // 伤害类型
    Item.width = 40; // 物品宽度
    Item.height = 40; // 物品高度
    Item.useTime = 20; // 物品使用时间
    Item.useAnimation = 20; // 物品使用动画时间
    Item.useStyle = 1; // 物品的使用方式
    Item.knockBack = 6; // 物品的击退
    Item.value = 10000; // 物品的价值
    Item.rare = 2; // 稀有度
    Item.UseSound = SoundID.Item1; // 使用物品的音效
    Item.autoReuse = true; // 是否可以自动使用（可以一直安装左键使用）
}
C#

在 `AddRecipes` 方法中，你可以添加物品的合成配方。

// 添加合成配方
public override void AddRecipes()
{
    Recipe recipe = CreateRecipe();
    recipe.AddIngredient(ItemID.DirtBlock, 10); // 添加配方原料
    recipe.AddTile(TileID.WorkBenches); // 添加合成台
    recipe.Register(); // 注册配方
}
